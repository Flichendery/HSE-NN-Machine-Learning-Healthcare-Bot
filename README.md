# Телеграм-бот для предсказывания болезней по списку симптомов


## Как запустить
1. Клонировать репозиторий
2. Создать виртуальное окружение
3. Установить требуемые библиотеки:
```
pip install -r requirements.txt
```
4. В файл `.env` вставить токен:
```
BOT_TOKEN=<token>
```
5. Запустить файл `bot.py`
